## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced search filters with date ranges and file type restrictions
- Introduced document templates for consistent formatting across teams
- Implemented real-time collaboration indicators showing active editors

## Improvements

- Enhanced mobile responsiveness for document editing on tablets and phones
- Optimized loading times for large document collections by 40%
- Added keyboard shortcuts for common editing actions

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched new API with comprehensive endpoints for document management
- Added support for importing and exporting documents in multiple formats
- Introduced granular permission controls for fine-tuned access management

## Breaking Changes

- Updated authentication flow requiring API key rotation for enhanced security
- Changed default document visibility from public to private in new workspaces
- Modified webhook payload structure to include additional metadata

</Update>

- Added support for dark mode across all platforms
- Enhanced integration capabilities with popular development tools

## Bug Fixes

- Resolved search functionality returning incomplete results for large datasets
